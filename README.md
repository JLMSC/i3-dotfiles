# i3-dotfiles
My i3wm custom dotfiles.

## Dependencies 
These are being used at least once in the `config`, includes utilities too:
```
- CaskaydiaCove Nerd Font Mono
- dmenu
- alacritty
- thunar
- mate-system-monitor
- pulseaudio
- pavucontrol
- xinput
- xrandr
- feh
- xprop
```

## Paths
i3: `~/.config/i3`

Polybar: `~/.config/polybar`
